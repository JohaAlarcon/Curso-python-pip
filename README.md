# Game Project

Para inicializar el juego debes ejecutar las siguientes instrucciones en la terminal:

``` sh
cd game 
python3 main.py
```

# App Project

Para inicializar el juego debes ejecutar las siguientes instrucciones en la terminal:

``` sh
git clone
cd app
source/env-app/bin/activate
pip3 install -r requirements.txt 
python3 main.py
```
